# API для витрины проектов ИШИТР ТПУ

API предоставляет данные о студентах, проектах, командах и рейтингах для фронтенда витрины ИШИТР ТПУ.

---

## Общая информация

| Параметр          | Значение                |
|-------------------|-------------------------|
| Базовый URL (dev) | `http://127.0.0.1:8000` |
| Формат ответа     | JSON                    |
| Кодировка         | UTF-8                   |
| HTTP методы       | GET (только чтение)     |

---

## Сводка эндпоинтов

| #  | Метод | URL                               | Описание                     |
|----|-------|-----------------------------------|------------------------------|
| 1  | GET   | `/api/students/`                  | Список всех студентов        |
| 2  | GET   | `/api/students/{login}/`          | Детали студента              |
| 3  | GET   | `/api/students/{login}/teams/`    | Команды студента             |
| 4  | GET   | `/api/students/{login}/activity/` | Активность студента          |
| 5  | GET   | `/api/students/{login}/medals/`   | Медали студента              |
| 6  | GET   | `/api/projects/`                  | Витрина проектов             |
| 7  | GET   | `/api/projects/{id_project}/`     | Детали проекта               |
| 8  | GET   | `/api/teams/`                     | Список команд                |
| 9  | GET   | `/api/teams/{team_id}/`           | Детали команды с участниками |
| 10 | GET   | `/api/activity/`                  | Все записи активности        |
| 11 | GET   | `/api/leaderboard/students/`      | Рейтинг студентов (топ-10)   |
| 12 | GET   | `/api/leaderboard/projects/`      | Рейтинг проектов             |

## 1. Студенты

### 1.1. GET /api/students/

**Описание:** Возвращает список всех студентов.

**Используемые поля:**

| Поле               | Тип     | Описание                          |
|--------------------|---------|-----------------------------------|
| `login`            | string  | Уникальный идентификатор студента |
| `full_name`        | string  | ФИО студента (вычисляемое)        |
| `student_group`    | string  | Учебная группа                    |
| `direction_name`   | string  | Направление подготовки            |
| `study_year`       | integer | Курс                              |
| `faculty`          | string  | Факультет                         |
| `study_score`      | decimal | Средний балл                      |
| `top_view`         | string  | Статус (лидер/активный/новичок)   |
| `history_work_all` | decimal | Всего часов работы                |

**Неиспользуемые поля:**

| Поле                 | Причина                                              |
|----------------------|------------------------------------------------------|
| `someone_id`         | Внутренний ID для интеграции с API вуза              |
| `first_name`         | Заменено на вычисляемое поле `full_name`             |
| `last_name`          | Заменено на вычисляемое поле `full_name`             |
| `patronymic`         | Заменено на вычисляемое поле `full_name`             |
| `debt_count`         | Скрыто в общем списке (конфиденциальная информация)  |
| `history_work_sem`   | Скрыто в общем списке (только на детальной странице) |
| `history_work_week`  | Не используется в текущей версии интерфейса          |
| `history_work_month` | Скрыто в общем списке (только на детальной странице) |

---

**Параметры пути:**

| Параметр | Тип    | Описание       |
|----------|--------|----------------|
| `login`  | string | Логин студента |

**Используемые поля (все поля модели, кроме неиспользуемых):**

| Поле                 | Тип     | Описание                                |
|----------------------|---------|-----------------------------------------|
| `login`              | string  | Уникальный идентификатор студента       |
| `full_name`          | string  | ФИО студента (вычисляемое)              |
| `student_group`      | string  | Учебная группа                          |
| `direction_name`     | string  | Направление подготовки                  |

| `study_score`        | decimal | Средний балл                            |
| `debt_count`         | integer | Количество академических задолженностей |
| `top_view`           | string  | Статус (лидер/активный/новичок)         |
| `history_work_sem`   | decimal | Часы работы за текущий семестр          |
| `history_work_all`   | decimal | Всего часов работы                      |

**Неиспользуемые поля:**

| `team_info.period_start` | Скрыто на витрине (только на детальной странице) |
| `team_info.period_end`   | Скрыто на витрине (только на детальной странице) |

## 5. Рейтинг (Leaderboard)

### 5.1. GET /api/leaderboard/students/

**Описание:** Возвращает топ-10 студентов по часам работы.

**Используемые поля:**

| Поле                 | Тип     | Описание                        |
|----------------------|---------|---------------------------------|
| `login`              | string  | Логин студента                  |
| `full_name`          | string  | ФИО студента                    |
| `student_group`      | string  | Учебная группа                  |
| `top_view`           | string  | Статус (лидер/активный/новичок) |
| `history_work_all`   | decimal | Всего часов работы              |
| `history_work_sem`   | decimal | Часы за семестр                 |
| `history_work_month` | decimal | Часы за месяц                   |
| `study_score`        | decimal | Средний балл                    |
| `total_medals`       | integer | Общее количество медалей        |

**Неиспользуемые поля:** Отсутствуют (сериализатор отдаёт только нужные для рейтинга поля).

---

### 5.2. GET /api/leaderboard/projects/

**Описание:** Возвращает рейтинг проектов по суммарным часам работы.

**Используемые поля:**

| Поле            | Тип     | Описание                              |
|-----------------|---------|---------------------------------------|
| `id_project`    | integer | ID проекта                            |
| `project_name`  | string  | Название проекта                      |
| `total_hours`   | float   | Суммарные часы работы всех участников |
| `members_count` | integer | Количество участников в команде       |
| `team_id`       | integer | ID команды проекта                    |

**Неиспользуемые поля:** Отсутствуют (сериализатор отдаёт только нужные для рейтинга поля).

## 6. Коды ответов

| Код | Описание                  |
|-----|---------------------------|
| 200 | Успешный запрос           |
| 404 | Объект не найден          |
| 500 | Внутренняя ошибка сервера |

---

## 7. Примечания

- Все эндпоинты доступны **только для чтения** (GET).
- Поля `full_name` и `student_full_name` являются вычисляемыми (формируются из `last_name`, `first_name`, `patronymic`).
- Поле `rol` в командах принимает значения: `Студент`, `Наставник`.
- Поле `top_view` принимает значения: `лидер`, `активный`, `новичок`.
- В эндпоинте `/api/leaderboard/students/` возвращается **топ-10** студентов.
- В эндпоинте `/api/leaderboard/projects/` проекты сортируются по `total_hours` (сумма часов всех участников).
